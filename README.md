# Log Parser with Integrated AbuseIPDB API Check

A small security tool that parses log files and checks suspicious IPs against the AbuseIPDB database.
Its a personal learning project created with a goal to develop practical skills in cybersecurity, Python scripting and security log analysis

## Functionality
- Parses log files based on alert type selected by the user
- Stores found IPs and their frequency in a dictionary
- Queries the AbuseIPDB API for reputation data on discovered IPs
